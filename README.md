RAPPORT D'ANALYSE : OCCUPATION DES PARKINGS DE MONTPELLIER

Introduction Ce projet présente une étude automatisée de la mobilité urbaine à Montpellier, réalisée dans le cadre de la SAE 15. L'objectif est de transformer des données brutes issues de l'Open Data en informations stratégiques pour la municipalité.

Synthèse des Résultats Le succès de ce projet réside principalement dans la transformation des données brutes en une preuve visuelle claire des dynamiques d’activité sociale de Montpellier. Les résultats obtenus valident techniquement les hypothèses sur l’activité hebdomadaire : les creux nocturnes minimaux des lundis, la stabilité des jours de semaine, les pics nocturnes élevés le vendredi et le samedi, et le repos commercial marqué du dimanche à midi.

Concernant l'intermodalité, le système de collecte Cloud s’est concentré sur la base de données véhicules. Cependant, l’observation ponctuelle des stations comme Comédie et Gare Saint-Roch montre que la saturation des véhicules n’a pas de relation linéaire directe avec l’usage des vélos. Cela suggère que l’utilisateur du centre historique privilégie le tramway comme connexion principale.

Ce travail démontre que l’analyse automatisée est un outil déterminant pour optimiser la mobilité urbaine d’une ville intelligente.
